1. Download SQL files for books and .csv files for movies
2. Create postresql database
3. In postgresql, create books, users and book_ratings relational tables
4. Create python virtual enviroment 'python -m venv .venv'
5. Activate virtual enviroment '.\.venv\Scripts\activate.bat'
6. Install requirements 'pip install -r requirements.txt'
7. Change load_sql.py to query desired data from postgresql and save them to pandas DataFrames
8. Run main.py to get the recommendations
